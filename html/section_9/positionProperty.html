static

kalau static mau di positioning kayak gimanapun dia gak akan berubah dan ini adalah default value dari properti position pada tiap element

relative

top left balala2 artinya dia ngasih space untuk di situ, jd kalau kita kasih top 100px, di bakal turun kebawah 100px

absolute

sumpa bingung ini apa, jd posisinya tidak exist di flow document, posisinya relatif terhadap body, kecuali kalau parrent elementnya di set position relative, dia bakal relative sama parrentnya, kyak di bawha gini

fixed

dia gak beda jauh sama absolute cuma dia gak punya hubungan sama sekali dengan parrent element. dia pure offset dengan initial containing block
